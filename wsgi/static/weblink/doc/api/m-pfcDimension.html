Library | Module

Module "pfcDimension"

Description

Contains interfaces related to solid model dimensions.
Enumerated Types
enumpfcDimensionType
This enumerated type lists the possible dimension types.
enumpfcDimToleranceType
This enumerated type lists the possible dimension tolerance types.
enumpfcToleranceTableType
This enumerated type lists the possible tolerance table types.

Classes
classpfcBaseDimension
This class defines a base dimension, which can be a dimension or reference dimension.
classpfcDimension
This class describes a dimension.
classpfcDimensionShowInstructions
The base class for instructions objects containing information about where a dimension should be shown.
classpfcDimTolerance
This class defines a dimension tolerance.
classpfcDimTolISODIN
This class displays the tolerance table type, tolerance table and column for the dimension tolerance if the tolerance table type is hole or shaft.
classpfcDimTolLimits
This class displays the limits for the dimension tolerance, for example, a dimension with upper limit of 50.1 and a lower limit of 49.9.
classpfcDimTolPlusMinus
This class displays the dimension tolerance in the form +plus value and -minus value.
classpfcDimTolSymmetric
This class displays the dimension tolerance in symmetrical form (+ or - the value).
classpfcDimTolSymSuperscript
This class displays the dimension tolerance in symmetrical form (+ or - the value). The text of the tolerance is displayed in a superscript format with respect to the dimension text.
classpfcRefDimension
This class describes a reference dimension.

Collections
classpfcBaseDimensions
This interface contains a sequence of dimension and reference dimension objects.
