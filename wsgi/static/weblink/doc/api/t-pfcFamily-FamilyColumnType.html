Enum pfcFamilyColumnType

Description

This enumerated type contains the possible types of family table columns.



Values

A user-defined parameter column, represented by a pfcFamColParam object.
FAM_DIMENSION
A dimension column, represented by a pfcFamColDimension object.
FAM_IPAR_NOTE
A parameter note column, represented by a pfcFamColIParNote object.
FAM_FEATURE
A feature column, represented by a pfcFamColFeature object.
FAM_ASMCOMP
A component feature column, represented by a pfcFamColComp object.
FAM_UDF
A UDF group column, represented by a pfcFamColUDF object.
FAM_ASMCOMP_MODEL
A component model column, represented by a pfcFamColCompModel object.
FAM_GTOL
A geometric tolerance column, represented by a pfcFamColGTol object.
FAM_TOL_PLUS
A dimension tolerance plus value column, represented by a pfcFamColTolPlus object.
FAM_TOL_MINUS
A dimension tolerance minus value column, represented by a pfcFamColTolMinus object.
FAM_TOL_PLUSMINUS
A dimension tolerance plus/minus symmetric value column, represented by a pfcFamColTolPlusMinus object.
FAM_SYSTEM_PARAM
A system parameter column, represented by a pfcFamColSystemParam object.
FAM_EXTERNAL_REFERENCE

FAM_QUILT_PARAM

FAM_ANNOT_ELEM_PARAM
