Description

CURVE_POINT
A point
CURVE_LINE
A line
CURVE_ARROW
An arrow
CURVE_ARC
An arc
CURVE_SPLINE
A spline curve
CURVE_BSPLINE
A Non-Uniform Rational B-Spline
CURVE_CIRCLE
A circle
CURVE_ELLIPSE
An elliptical curve
CURVE_POLYGON
A polygon
CURVE_TEXT
A text entity
CurveType_nil
