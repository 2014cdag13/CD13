Library | Module

Class pfcSuppressOperation


Description

This class defines a suppress operation.
Direct Parent Classes:
pfcFeatureOperation


Property Summary

booleanAllowChildGroupMembers
If true the children of feature, if members of a group, will be individually suppressed from their group. If false, then the group containing the feature's children will be also suppressed. This attribute can be set to true if and only if both pfcSuppressOperation.Clip and pfcSuppressOperation.AllowGroupMembers are set to true. Default value is false.
booleanAllowGroupMembers
If true the feature, if member of a group, will be individually suppressed from the group. If false, the group containing the feature will be also suppressed. This attribute can be set to true if and only if pfcSuppressOperation.Clip is also set to true. Default value is false.
booleanClip
A Boolean flag that specifies whether to suppress the selected feature and all the features that follow. Default value is false.
