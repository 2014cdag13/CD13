Parameters:
Params
 
Returns:
The function returns one of the following values:
  • PLACE_ON_BOUNDARY -- The point is on the contour.
  • PLACE_OUTSIDE -- The point is outside the contour.
  • PLACE_INSIDE -- The point is inside the contour.
